# Schematics Directory

This directory contains circuit diagrams, wiring diagrams, and other technical drawings for the LED Hat Small project.

## Contents

- Circuit diagrams (.pdf, .png, .svg)
- Wiring diagrams
- Layout drawings
- 3D models (if available)
- PCB layouts (if applicable)

## File Formats

- **PDF**: For printable schematics
- **PNG/SVG**: For web viewing
- **KiCad/Eagle**: For editable circuit designs
- **STEP/STL**: For 3D models

## Adding Schematics

When adding new schematics:
1. Use clear, descriptive filenames
2. Include version numbers if multiple versions
3. Add notes/annotations for clarity
4. Update this README with description

## Schematic Types

### Circuit Diagrams
- Overall system schematic
- Power distribution
- Signal routing
- Component connections

### Wiring Diagrams
- Physical wiring layout
- Cable routing
- Connection points
- Wire colors/gauge

### Layout Drawings
- Component placement
- Physical dimensions
- Mounting locations
- Clearance requirements
