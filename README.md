# LED Hat Small - Pixelblaze Wearable Project

A wearable LED hat project powered by Pixelblaze, featuring dynamic patterns and WiFi control.

## Project Structure

├── README.md              # This file
├── docs/                  # Project documentation

│   └── user-guide.md     # User guide and operation
├── firmware/             # Pixelblaze patterns and code
│   ├── patterns/         # Pattern files (.pe)
│   ├── presets/          # Pattern presets and configurations
│   └── README.md         # Firmware documentation
├── hardware/             # Hardware documentation

## Quick Start

1. **Hardware Setup**: See `hardware/assembly.md` for assembly instructions
2. **Firmware Upload**: See `firmware/README.md` for pattern installation
3. **Operation**: See `docs/user-guide.md` for usage instructions

## Features

- WiFi-enabled pattern control via Pixelblaze
- Multiple pre-programmed patterns
- Real-time pattern switching
- Customizable brightness and speed
- Battery-powered operation

## Requirements

- Pixelblaze controller board
- WS2812B (or compatible) LED strip/matrix
- Power supply (battery pack recommended)
- WiFi network for web interface
