# User Guide

## Getting Started

### Initial Setup

1. **Power On**
   - Connect battery or power supply
   - Wait for Pixelblaze to boot (LED indicator will show status)

2. **Connect to WiFi**
   - Pixelblaze creates its own WiFi network (default: "Pixelblaze")
   - Connect your device to this network
   - Open browser and navigate to: `http://192.168.4.1` or `http://pixelblaze.local`

3. **Configure Network (Optional)**
   - Go to Settings â†’ WiFi
   - Connect to your home network for remote control
   - Note the IP address assigned

### Basic Operation

#### Pattern Selection
- Use the web interface to browse available patterns
- Click on a pattern to activate it
- Patterns will start immediately

#### Pattern Controls
- **Brightness**: Adjust overall brightness (0-255)
- **Speed**: Control pattern animation speed
- **Color**: Modify color schemes (pattern-dependent)
- **Save Preset**: Save current settings as a preset

#### Pattern Switching
- **Manual**: Select from pattern list in web interface
- **Auto-Play**: Enable pattern rotation with timer
- **Random**: Randomly cycle through patterns

## Advanced Features

### Creating Custom Patterns
1. Navigate to Pattern Editor in web interface
2. Use Pixelblaze Expression language to create patterns
3. Test patterns in real-time
4. Save patterns to flash memory

### Pattern Presets
- Create presets for different occasions
- Quick-switch between preset configurations
- Share presets with other Pixelblaze users

### Remote Control
- When connected to WiFi network, access from any device
- Use mobile app (if available) for quick control
- Set up automation and scheduling

## Troubleshooting

### No WiFi Network Visible
- Check power connection
- Wait 30 seconds for boot sequence
- Reset Pixelblaze if needed (see hardware notes)

### Patterns Not Loading
- Check flash memory space
- Verify pattern file format (.pe)
- Try uploading pattern again

### Battery Issues
- Monitor battery level via web interface
- Reduce brightness to extend battery life
- Check battery connections

### LED Display Issues
- Verify data connection to first pixel
- Check power supply voltage
- Test with known-good pattern

## Maintenance

### Regular Maintenance
- Clean LED surface with soft cloth
- Check battery connections monthly
- Update firmware periodically

### Storage
- Store in cool, dry place
- Disconnect battery for long-term storage
- Keep at 50% charge for storage

## Safety

- Do not exceed maximum brightness for extended periods
- Avoid exposing to water unless rated for it
- Do not modify power connections while powered
- Follow battery safety guidelines
