# Project Overview

## Introduction

This project is a wearable LED hat powered by a Pixelblaze controller, designed to create dynamic, eye-catching light patterns for events, performances, or personal expression.

## Goals

- Create a compact, wearable LED display
- Provide easy pattern customization via Pixelblaze web interface
- Ensure comfortable, long-lasting battery operation
- Maintain a clean, professional appearance

## Key Features

### Hardware
- Compact form factor suitable for hat integration
- Efficient power management for extended battery life
- Durable construction for active use
- Weather-resistant design (optional)

### Software
- WiFi-enabled pattern control
- Real-time pattern switching
- Custom pattern creation via Pixelblaze web interface
- Pattern presets for quick setup

## Use Cases

- Festivals and events
- Performances and shows
- Personal expression and art
- Group coordination and identification
- Safety and visibility
